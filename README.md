Repository for my Portfolio/Blog website.

The plan is to push updates to the repository, then have a Pipeline automatically push changes to my VPS directly, when updated.

Currently, this is made with Hugo (https://gohugo.io/) and I am using the PaperModX Theme (https://github.com/reorx/hugo-PaperModX).

The theme is placed as a submodule, for the moment, but I am making changes to the partial HTML files, so will be looking at pulling this seperately, and modifying further, for my own needs.
